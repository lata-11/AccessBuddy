ACCESS BUDDY

Introduction

Welcome to Access Buddy, a platform designed to enhance accessibility and provide support for individuals with disabilities. Our goal is to create tools and resources that make everyday tasks easier and more inclusive. This guide will walk you through the main features of the application and how to use them.

Navigating the Interface

1. Home Screen

  • Profile Section: Displays your profile picture and name.
  • Task Management: Add, remove, and manage your tasks.
  • Study Help: Access features for reading documents.
  • Emergency Contact: Manage and view emergency contacts.
  • Account Settings: Access settings, change your profile picture, and log out.

2. Task Management

Add/Remove Task:

  • Add Task: Type your task into the input field and press "Add Task" to save it.
  • Remove Task: Click the "Remove" button next to any task to delete it.
  • Voice Commands: Click the "Voice Command" button to use voice commands for adding or removing tasks. Simply speak your command, and the app will handle it.

Read All Tasks:

  • Button: Click the "Read All Tasks" button to have the app read out all your current tasks.

3. Reminder Management

Add Reminder:

  • Form: Fill in the reminder message, date, time, and phone number in the form.
  • Save Reminder: Press "Save Reminder" to add it to your list. The reminder will be saved and displayed.
  • Voice Commands: Use the voice command button to add reminders by speaking.

4. Study Help

Read Document:

  • Document Upload: Upload a document for the app to read.
  • Voice Reading: The app will read out the document content using text-to-speech.

5. Accessibility Features

Text-to-Speech:

  • Text Area: Type text into the provided text area.
  • Read Button: Press "Read" to have the app read the text aloud using speech synthesis.

6. Dropdown Menus

  • Task Management Dropdown: Contains options to add/remove tasks and set reminders.
  • Study Help Dropdown: Contains options for document reading.
  • Emergency Contact Dropdown: View and manage emergency contacts.
  • Account Settings Dropdown: Access app guide, change profile picture, and log out.

Voice Commands

General Commands:

  • Add Task: Speak “Add [task description]” to add a task.
  • Remove Task: Speak “Remove [task description]” to delete a task.
  • Exit: Speak “Exit” or “Bye” to stop voice commands.
